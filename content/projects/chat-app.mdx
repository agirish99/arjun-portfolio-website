A modern chat application that enables real-time messaging between users with features like typing indicators and message history.

- Implemented real-time messaging using Socket.io
- Added user authentication with JWT tokens
- Created typing indicators and online status
- Built message persistence with MongoDB
- Added file upload support for images and documents
- Implemented responsive design for mobile and desktop
- Added message search and filtering capabilities
