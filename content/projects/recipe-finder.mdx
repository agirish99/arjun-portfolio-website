A full-stack recipe application that allows users to search, filter, and save recipes based on ingredients and dietary preferences.

- Built with Next.js for server-side rendering and optimal performance
- Integrated Spoonacular API for comprehensive recipe database
- Implemented user authentication and recipe bookmarking
- Added advanced filtering by cuisine, diet, and cooking time
- Created a responsive design with mobile-first approach
- Used Prisma ORM for database management and type safety
